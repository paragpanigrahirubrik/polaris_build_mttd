MTTD Data

Overview of Mean Time to Detect (MTTD) data for failed polaris master builds.

This tracks time from time of commit for offending PR to build failure. If offending PR is not found, we fallback to the oldest commit being built in the job

There might be some builds which don't have any new commit being built. In that case the offending PR/commit field is N/A and TTD is just the build time.
